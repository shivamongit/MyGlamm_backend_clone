## [](https://github.com/shivamongit/MyGlamm_backend_clone.git#run-locally)Run Locally

Clone the project

-[git clone](https://github.com/shivamongit/MyGlamm_backend_clone.git)

Go to the project directory

Install dependencies

  npm install

Start the server

  npm start
